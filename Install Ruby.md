# Install Ruby

[Follow Ruby installation tutorial](https://gorails.com/setup/ubuntu/14.04) using `rvm` option. Missing libraries were [libgmp-dev](https://github.com/copiousfreetime/hitimes/issues/48)
	
	apt-get install libgmp-dev
